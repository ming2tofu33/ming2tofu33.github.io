Wine Quality Prediction

Machine Learning

와인의 11가지 화학적 속성을 기반으로 품질 등급(3~8점)을 예측하는 분류 모델 구축 프로젝트.

  • 최적 모델 선정: RandomForest 모델을 채택하여 85.2%의 정확도 달성.
  • 성능 최적화: 탐색적 분석(EDA) 기반 특성 엔지니어링 및 하이퍼파라미터 튜닝을 통한 분류 성능 향상.
  • 주요 특성 분석: 와인 품질에 가장 큰 영향을 미치는 알코올 도수(alcohol)와 휘발성 산도(volatile_acidity) 등 핵심 특성 도출.
  • 핵심 역량: 데이터 전처리부터 모델 비교, 최적화까지 머신러닝 분류 프로세스 전반에 대한 숙련도 확보.

SQL Music Database Analysis

Database Analysis

  • 핵심 역량: JOIN, Subquery, Window Function 등 고급 SQL 기술을 활용하여 비즈니스 인사이트를 도출하는 실무 분석 역량 확보.
